ADMS Project - Front End (Version 1)

### Project Description
A streamlined web-based interface for the Activity Data Management System (ADMS). This version focuses on providing a structured data entry form and a clean table display for managing records.

### Tech Stack
* **HTML5**: For page structure and forms.
* **CSS3**: For layout and basic styling.

### Installation & Usage
1. Download or clone this repository to your local machine.
2. Navigate to the project folder.
3. Open `index.html` in any web browser to view the application.
